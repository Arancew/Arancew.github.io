用于甚高分辨率遥感目标检测的分层鲁棒卷积神经网络

Hierarchical and Robust Convolutional Neural Network for Very High-Resolution Remote Sensing Object Detection

用于甚高分辨率遥感目标检测的分层鲁棒卷积神经网络

主要贡献

1.构建了一个新的CNN框架。该框架在不需要复杂结构的情况下极大地促进了定位和识别

2.提出了一种高速定位结构。该结构利用了分层卷积层，并考虑了空间分辨率和语义信息

3.在新框架中实现了RSRE结构，以提高识别精度。该结构同时处理旋转和缩放的变化因子，无需任何额外参数

4.新的数据集，是以往的10倍大小

Abstract

深度学习在遥感目标检测方面，尤其是基于卷积神经网络（CNN）的深度学习已逐渐获得竞争优势，常规方法忽略了局部信息，而局部信息可以提供空间信息，有助于准确定位，为了解决这个问题，提出了一种层次结构鲁棒卷积神经网络

  • 提取多尺度卷积特征来表示层次空间语义信息
  • 将多个全连接层特征堆叠在一起，以提高旋转和缩放鲁棒性（旋转和缩放等可变因素会影响RSIs中的目标检测精度）

一、INTRODUCTION

目标检测的目的包括目标定位和目标识别，分别生成目标的位置和类别。

传统方法是手动提取特征，神经网络可以提取高级特征，然后介绍了写VGG等经典CNN网络

A. Developments of the Localization in RSI Object Detection

RSI目标检测中定位技术的发展

就是先把物体找出来，然后再识别这个物体是什么

B. Developments of the Recognition in RSI Object Detection

RSI目标检测中识别技术的发展

  1. 传统方法，Sedaghat等人引入了一种对旋转、缩放和平移不变性的特征，解决了不同分辨率，物体不同方向的问题

  2. Cheng等人提出的基于CNN的RSI目标检测模型，通过提出旋转不变层来处理旋转问题

  3. Hu等人考虑对一组多尺度图像进行卷积特征编码，以解决RSI场景分类中的缩放问题

  4. 本文通过添加旋转和缩放鲁棒结构来解决旋转和放缩的问题，该结构根本不增加参数

C. Data Sets in RSI Object Detection

完善了一个数据集，包含2w张图片和13个类别，每个类比有4w个注释对象，各类之间保持平衡

二、RELA TED WORK

传统方法

1.SIFT，尺度不变特征

2.HOG，方向梯度直方图

3.Bag-of-words (BoW) 模型，用途比较广泛

4.稀疏编码，开发一些纹理特征

神经网络方法

介绍了下什么是神经网络，什么是CNN

提前人工作只是解决放缩和旋转，但没有同时解决

本文提出的HRCNN同时解决了上面的问题

三、HRCNN FOR REMOTE SENSING OBJECT DETECTION

利用AlexNet进行预训练，HSS结构采用了特征金字塔（将不同的语义结合起来会更好）

旋转的放缩

航空或卫星图像中，视角是从天空向下到地面的。因此，物体似乎更可能躺在图像上，而不是站在地面上，因此需要旋转不变性，RSI可能具有不同的空间分辨率。第二，同一类别的物体大小也不同，这就要求需要尺度不变性。

然后统统用全连接层搞定

CNN提取特征

SVM进行分类

因为是非线性的，所以引入的核函数

回归器用来修订边界框

四、EXPERIMENTS

原有的数据集不足以训练神经网络，因此构建了一个新的数据集

为了保证子集的平衡，采用了一种分类算法（就是划分训练集和测试集）

评价了一下召回率和正确率

细节方面

CNN用的就是ALexNet

需要调参的全连接层，学习率是动态的

后续用支持向量机来取代softmax回归来提高精度
